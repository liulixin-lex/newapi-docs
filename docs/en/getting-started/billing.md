# Buy Credits

::: tip Entry Point
After signing in to the console, open the `Wallet` page from the left sidebar to buy credits.
:::

## Top-up Steps

You can add credits to your account in two ways: top up the balance directly, or buy a redemption code and redeem it back on the Wallet page.

### Method 1: Top Up Balance

1. In `Add funds`, choose a fixed amount, or enter the amount you want to top up under `Custom amount`.
2. Confirm the `Amount due` at the bottom of the page, then choose a payment method and pay.

![Wallet page for choosing a top-up amount, payment method, and amount due](/images/tutorial/6.webp)

### Method 2: Buy a Redemption Code

1. Click `Get it here` below to open the redemption-code purchase website.

![Wallet page showing the entry to get a redemption code](/images/tutorial/101.webp)

2. Select the credit package you want to buy and complete the purchase.

![Redemption code purchase page for choosing a top-up package](/images/tutorial/102.webp)

3. Copy the redemption code you purchased and return to the Wallet page.

4. Paste it into the redemption box to redeem the equivalent credits.

![Wallet page for pasting a redemption code and redeeming credits](/images/tutorial/104.webp)
