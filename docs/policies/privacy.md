# 四、隐私政策与数据处理说明

## 1. 基本原则

GGUUAI 尊重并保护用户隐私和数据安全。我们会按照合法、正当、必要、诚信、目的明确、最小必要和安全保护原则处理个人信息和相关数据。

本隐私政策说明 GGUUAI 在提供 API 中转、模型聚合、控制台、计费、风控、客服和相关服务时可能收集、使用、保存、共享、转移、删除和保护的信息。

## 2. 我们可能收集的信息

为提供和维护服务，GGUUAI 可能收集以下信息：

1. 账号信息：用户名、邮箱、登录方式、认证信息、组织信息、团队信息、角色权限、联系信息；
2. 支付与订单信息：订单号、充值金额、支付状态、支付渠道、退款记录、发票或账务信息、必要的付款或收款证明；
3. API 调用元数据：请求时间、模型名称、用量、消耗、响应状态、错误码、请求 ID、路由信息、延迟、并发、速率限制、缓存状态；
4. 安全与风控信息：IP 地址、设备信息、浏览器或客户端信息、登录记录、异常调用记录、密钥使用记录、风控审核记录；
5. 客服与沟通信息：您向我们提交的问题、反馈、申诉、工单、邮件、附件和处理记录；
6. 用户内容：您或您的最终用户通过 API 提交的输入、文件、上下文、参数、输出或相关数据，但仅在提供服务、路由请求、计费、排障、安全、合规、用户主动提交、用户开启日志功能或法律法规要求所需范围内处理。

## 3. 我们如何使用信息

GGUUAI 可能将相关信息用于以下目的：

1. 创建、维护和管理账户；
2. 提供 API 中转、模型聚合、模型路由、控制台和相关功能；
3. 进行身份验证、权限控制、API Key 管理和安全保护；
4. 计算用量、扣费、充值、退款、账务核对和订单管理；
5. 排查故障、优化服务、分析错误、提升稳定性；
6. 识别、预防和处理滥用、攻击、欺诈、盗刷、拒付、洗钱、套现、异常退款、规避限制和其他风险；
7. 执行使用政策、服务条款、上游要求和适用法律法规；
8. 响应客服、投诉、申诉、隐私请求和安全事件；
9. 发送服务通知、条款更新、价格变更、维护公告和安全提醒；
10. 履行法律法规、监管机关、司法机关、支付渠道或上游服务商的合规要求。

## 4. API 内容与模型训练

默认情况下，GGUUAI 不使用用户通过 API 提交的输入和输出训练通用模型。

如 GGUUAI 后续提供开发合作、数据授权、产品改进、模型评测、反馈数据使用、微调、模型定制或类似计划，只有在您明确选择加入、授权或签署相关协议后，GGUUAI 才会按照对应条款使用您授权的数据。

您理解并同意，GGUUAI 作为 API 中转和模型聚合服务，可能需要将您的请求、输入、参数、文件或上下文发送至您选择或平台路由的上游模型服务商，以便完成模型调用。不同上游服务商可能有不同的数据保留、审核、训练、日志、安全和隐私政策。上游服务商对数据的处理以其适用条款、政策、模型规则和您的配置选择为准。

除为提供服务、路由请求、计费、排障、安全、风控、合规、法律要求或用户配置所需外，GGUUAI 不会主动将您的 API 输入和输出用于训练通用模型。

## 5. 数据保存期限

GGUUAI 将在实现处理目的所需的最短期限内保存相关信息，法律法规、监管要求、支付渠道、争议处理、安全审计或上游服务规则要求更长期限的除外。

1. 账号信息：账户存续期间保存；注销后在法律允许范围内删除或匿名化，但为账务、争议、安全和合规所需的信息可继续保存必要期限；
2. 订单、充值、退款和账务记录：保存不少于 6 个月。
3. API 调用元数据：不超过 90 天
4. 错误日志、排障日志和安全日志：不超过 90 天；涉及安全事件、争议、欺诈、违规或监管要求的可延长。
5. 用户主动提交的客服、申诉和证明材料：处理完成后不超过 1 年，法律另有要求除外。
6. API 输入和输出正文：默认不长期保存；如用户开启日志、缓存、微调、文件、知识库或排障功能，则按对应功能说明保存。

## 6. 信息共享、委托处理与第三方

为提供本服务，GGUUAI 可能在必要范围内向以下第三方提供、共享或委托处理相关信息：

1. 上游模型服务商，用于完成模型调用、返回输出、执行内容安全或模型能力；
2. 云平台、网络服务商、CDN、存储、日志、监控和安全服务商，用于托管、传输、存储、排障和安全保护；
3. 支付渠道、银行、收款服务商、发票或账务服务商，用于充值、退款、对账、发票和风险审核；
4. 风控、反欺诈、安全、内容安全和合规服务商，用于识别异常、攻击、欺诈、滥用和违规；
5. 客服、工单、邮件或沟通工具，用于处理咨询、投诉、申诉和通知；
6. 监管机关、司法机关、执法机关、仲裁机构、上游服务商或其他依法有权主体，在法律法规、监管要求、司法程序、上游强制要求或保护合法权益所必要时提供。

我们会要求第三方在必要范围内处理相关信息，并采取合理安全措施。但第三方服务可能有其独立条款、隐私政策和数据处理规则，您应在使用相关模型、支付渠道或第三方能力前自行了解并遵守。

## 7. 跨境与跨地区处理

GGUUAI 面向法律允许、支付渠道支持、上游服务政策允许且平台风控评估通过的地区提供服务。为提供模型调用、云托管、网络传输、支付结算、风控、安全、客服或技术支持，您的信息可能在法律允许的地区之间传输、存储或处理。

如相关处理涉及个人信息跨境或跨地区传输，GGUUAI 将在适用法律要求的范围内采取必要措施，包括但不限于告知处理目的、数据类型、接收方类别、保存期限、安全措施和用户权利行使方式。

如果您或您的最终用户所在地区对数据出境、跨境传输、行业数据、敏感信息或个人信息有特殊要求，您应在使用前自行确认本服务是否适合您的业务场景，并根据需要与 GGUUAI 签署额外数据处理协议。

## 8. 用户权利

在适用法律允许和要求的范围内，您可以通过 [support@gguuai.com](mailto:support@gguuai.com) 请求查询、更正、复制、删除您的个人信息，撤回授权，注销账户，限制或反对特定处理，或获取个人信息处理规则说明。

为保护账户安全，GGUUAI 可能要求您验证身份、提供账户信息、订单信息或其他必要材料。对于无法验证身份、影响他人权益、涉及安全风控、账务审计、争议处理、法律义务、监管要求或技术上无法实现的请求，GGUUAI 可能依法拒绝或延迟处理，并向您说明原因。

账户注销后，您可能无法继续访问账户、API Key、余额、历史记录、配置和相关服务。注销不影响注销前已经发生的订单、消费、违规、争议、赔偿、退款审核、合规审计和法律责任。

## 9. 数据安全

GGUUAI 将采取合理的技术和管理措施保护信息安全，包括访问控制、权限管理、传输安全、日志审计、密钥保护、安全监控、异常检测和最小必要访问等措施。

您也应采取必要安全措施，包括妥善保管 API Key、定期轮换密钥、避免在前端公开密钥、限制密钥权限、设置合理额度、开启必要访问限制、监控异常消耗并及时删除泄露密钥。

如果发生可能影响您权益的数据安全事件，GGUUAI 将按照适用法律法规要求采取补救措施，并通过合理方式向受影响用户或主管机关通知。
