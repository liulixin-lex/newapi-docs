# Model Groups

## View Models Available in Each Group

1. In the console, click `Model Marketplace` at the top to view groups and models.
2. After entering the Model Marketplace, the area marked in red on the left is the token group mentioned in [Create an API Token](/en/getting-started/api-token.html) under Quick Start. The right side shows the models available under that group.
3. The value like `x0.1` after the group name on the left is the model multiplier.
4. Click `Details` to view the exact group a model belongs to and the multiplier for that group.
