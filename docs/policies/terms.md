# 二、服务条款

## 1. 定义

“用户”或“您”指注册、访问或使用本服务的个人、公司、组织、团队、开发者或其他实体。

“最终用户”指通过您的产品、服务、应用、系统、网站、集成、API Key 或其他方式直接或间接使用本服务能力的个人或实体。

“服务”指 GGUUAI 提供的 API 中转、模型聚合、模型接入、路由、控制台、访问凭证管理、用量统计、计费充值、开发者文档、技术支持以及相关能力。

“API Key”或“访问凭证”指用于访问本服务的密钥、令牌、访问令牌、账号权限、团队权限、项目权限或其他身份凭据。

“输入”指您或您的最终用户通过本服务提交的提示词、文本、图片、音频、视频、文件、代码、参数、请求、上下文、训练材料、业务数据或其他内容。

“输出”指模型、接口、工具或服务基于输入生成、返回、展示、处理或推断形成的内容。

“用户内容”指输入和输出的统称。

“调用元数据”指与 API 调用有关但通常不直接构成提示词正文或模型输出正文的信息，包括请求时间、模型名称、用量、消耗、响应状态、错误码、IP 地址、设备或环境信息、请求 ID、延迟、路由信息、账单记录、风控记录和必要日志。

“余额”或“额度”指您预先购买或平台赠送的服务使用额度。余额仅用于购买或抵扣 GGUUAI 服务消耗，不等同于现金、存款、电子钱包余额、支付账户余额或可自由提现资产。

“上游服务商”指向 GGUUAI 或您提供底层模型、云平台、网络线路、支付渠道、内容安全、日志、监控、数据处理、客服或其他基础设施能力的第三方。
