# CC-Switch Guide

::: tip Recommended Method
If you do not want to edit configuration files manually, use CC-Switch first.
:::

## What Is CC-Switch?

**An all-in-one helper for Claude Code, Codex, and Gemini CLI.**

It centrally manages provider configurations, MCP servers, Skills extensions, and system prompts for Claude Code, Codex, and Gemini CLI.

With [CC-Switch](https://ccswitch.io/zh/), you can:

- Switch API configurations with one click, moving quickly between multiple API providers.
- Manage configurations visually through a graphical interface.
- Manage Model Context Protocol servers.
- Use the system tray menu for quick switching.
