# 计费规则

本站采用美元 `$` 计费。

## 汇率

1 美元 = 6.8 人民币

## 分组倍率

根据不同分组设置不同倍率档位：

倍率 = 1 按官方价格进行消耗。

倍率 < 1 享折扣：例如 0.5 倍 = 5 折，即消耗同等数量 token，所需费用为原价费用的 0.5 倍。

倍率 > 1 需加价：1.5 倍 = 额外支付 50% 费用，即消耗同等数量 token，所需费用为基础费用的 1.5 倍。

## 海外模型

::: tip 计费口径
由于采用美元计价，故本站海外模型只使用一种倍率，即分组倍率。
:::

例如：

::: info 示例说明
本站 0.1 倍率 = 官方价格 * 0.1。
:::

| GPT-5.5 | 输入 | 输出 | 缓存读取 |

| 本站1.5倍率 | 7.5$/M | 45$/M | 0.75$/M |

## 国产模型

::: tip 计费口径
由于本站采用美元计价，故国产模型使用：汇率 * 分组倍率。
:::

例如：

::: info 示例说明
本站 0.103 倍率 = 0.147（汇率）* 0.7（折扣倍率），即官方价格的 0.7 倍。
:::
