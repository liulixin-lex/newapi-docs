# 八、条款更新

GGUUAI 可根据法律法规、监管要求、上游模型服务商政策、支付渠道规则、行业实践、业务变化、技术变化或风险控制需要更新本条款与政策。

更新后的条款将在平台公布后生效，或按公告中指定的时间生效。对于重大不利变更，GGUUAI 将在合理可行范围内通过电子邮件、站内信、控制台通知、文档公告、网站公告或其他合理方式提前通知。因法律法规、监管要求、安全风险、支付风险、上游强制要求、制裁或出口管制要求而进行的更新，可在发布后立即生效。

若您在条款更新后继续使用服务，即视为接受更新后的条款。若您不同意更新后的条款，应停止使用本服务，并可按本条款约定申请处理未使用的真实充值余额。
