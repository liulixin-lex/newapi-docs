# 六、服务可用范围与地区政策

## 1. 基本口径

GGUUAI 热衷于服务法律允许、支付渠道支持、上游服务政策允许、云平台和网络环境可支持且平台风控评估通过的所有地区。

服务可用性可能受到适用法律法规、监管要求、制裁与出口管制规则、支付渠道、上游模型服务商政策、云平台区域限制、网络环境、身份审核、行业要求以及平台风控要求影响。

即使某一地区可以访问 GGUUAI 页面、控制台、文档或部分服务，也不代表所有模型、接口、支付方式、充值方式、退款方式、通道、上下文长度、工具能力、图片或文件能力、缓存能力、微调能力或其他服务能力均可在该地区使用。

## 2. 用户责任

您应自行确认您所在地区、您的组织所在地、您的最终用户所在地、您的付款方式所在地、您的业务实际运营地以及您的具体使用场景是否允许访问和使用 GGUUAI 服务。

您不得在适用法律法规、监管规则、制裁限制、出口管制、上游模型服务商政策或平台规则禁止的国家、地区、用途、行业、主体或对象中使用本服务。

您不得通过 VPN、代理、跳板、虚假身份、伪造地址、代付、代充、账号转让、共享密钥、隐藏真实主体、隐藏真实用途或其他方式规避国家、地区、支付、合规、制裁、出口管制或风控限制。

如您的业务面向多个国家或地区，您应确保您及您的最终用户均符合当地法律法规、行业规范、平台规则和上游模型服务商政策。

如您不确定某一地区、行业、主体或用例是否可用，应在使用前通过 GGUUAI 联系邮箱咨询。

## 3. 支付渠道与充值可用性

支付渠道、充值方式、退款方式、到账时间、手续费、汇率、银行处理、支付机构审核和风控规则可能因国家、地区、支付机构、银行、收款方式、付款主体或政策变化而不同。

某些地区可能无法使用部分支付方式、充值渠道、退款渠道或收款方式。

支付渠道可能根据其自身规则拒绝、延迟、冻结、退回、撤销或审查交易。GGUUAI 对支付渠道自身规则、银行处理、汇率变化、跨境支付限制或第三方手续费不承担责任。

如支付或退款需要补充订单、身份、付款、收款、用途或其他证明，您应按 GGUUAI 要求提供真实、准确、完整的信息。

## 4. 出口管制、制裁与禁止对象

您不得将 GGUUAI 服务直接或间接提供、出口、再出口、转让、销售或用于被适用法律法规、制裁规则、出口管制规则或监管机关禁止的国家、地区、实体、个人、行业或用途。

您不得使受制裁、被限制、被禁止或存在重大合规风险的个人、实体、组织、政府机关、军事主体或其他对象直接或间接访问本服务。

如 GGUUAI 合理认为继续向您提供服务可能违反适用制裁、出口管制、监管要求、支付渠道规则或上游政策，GGUUAI 可立即限制、暂停或终止服务。
