一键配置环境变量即可让 Claude Code、Codex CLI、Gemini CLI 接入 GGUUAI。

接入配置

一行配置，即刻接入

# 1. 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 2. 配置 GGUU AI 中转地址
export ANTHROPIC_BASE_URL="{{API_BASE_URL}}"
export ANTHROPIC_AUTH_TOKEN="sk-xxxxxxxxxx"

# 3. 验证安装
claude --version
# 1. 安装 Codex CLI
npm install -g @openai/codex

# 2. 配置 GGUUAI Provider
# ~/.codex/config.toml
model_provider = "gguuai"
model = "gpt-5.5"
[model_providers.gguuai]
name = "GGUUAI"
base_url = "{{API_V1_BASE_URL}}"
wire_api = "responses"
requires_openai_auth = true

# ~/.codex/auth.json
{ "OPENAI_API_KEY": "sk-xxxxxxxxxx" }

# 3. 验证安装
codex --version
# 1. 安装 Gemini CLI
npm install -g @google/gemini-cli

# 2. 配置 GGUU AI 中转端点
export GOOGLE_GEMINI_BASE_URL="{{API_BASE_URL}}"
export GEMINI_API_KEY="sk-xxxxxxxxxx"

# 3. 验证安装
gemini --version
