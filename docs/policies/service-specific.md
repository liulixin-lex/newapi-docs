您应自行确认所选模型、接口、上下文长度、能力范围、价格和使用限制适合您的业务场景，并遵守相关上游模型服务商的适用规则。

## 2. API 使用限制

您必须遵守 GGUUAI 展示、通知或系统实际执行的 API 使用限制，包括但不限于：

1. 速率限制：不得超过平台规定或系统执行的调用频率；
2. 并发限制：不得超过账户、模型、通道、项目或套餐对应的并发额度；
3. 配额限制：不得超过分配额度、套餐额度、模型额度或风控额度；
4. 上下文限制：不得提交超过模型或通道支持范围的上下文、文件或参数；
5. 缓存规则：如平台支持缓存，您应遵守缓存期限、范围、费用和清除规则；
6. 内容安全限制：不得提交或生成违反使用政策的内容；
7. 监控与审计：GGUUAI 可为安全、计费、风控、排障、合规和服务优化目的监控 API 使用情况；
8. 规避禁止：不得通过拆分请求、轮换账户、轮换 IP、伪造身份、伪造终端用户、隐藏真实用途或其他方式规避限制。
