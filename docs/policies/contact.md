# 七、投诉、申诉与联系我们

## 1. 适用场景

您可以通过 [support@gguuai.com](mailto:support@gguuai.com) 联系 GGUUAI 处理以下事项：

1. 账户、登录、API Key 或安全问题；
2. 充值、扣费、退款、订单或账务问题；
3. 服务故障、模型不可用、配置错误或技术支持；
4. 隐私请求、数据删除、账户注销或个人信息处理问题；
5. 违规处理、账户冻结、服务暂停或退款拒绝申诉；
6. 知识产权、侵权、违法内容、AUP 违规或安全漏洞举报；
7. 服务地区、合规、上游模型或高风险用途咨询。

## 2. 申诉材料

为便于处理，您提交申诉、退款、隐私请求或投诉时，应尽量提供：

1. 账户邮箱或用户 ID；
2. 订单号、请求 ID、API Key 后四位或相关时间；
3. 问题描述和期望处理方式；
4. 相关截图、日志、付款凭证、业务用途说明或合规证明；
5. 可联系您的邮箱或其他联系方式。

GGUUAI 可根据安全、风控、支付、隐私或合规需要要求您补充材料。若您拒绝提供必要材料，或材料无法证明您的请求合理性，GGUUAI 可能无法处理或可能拒绝相关请求。

## 3. 处理原则

GGUUAI 将按合理顺序处理请求。普通咨询、技术支持和一般退款请求通常将在 3 个工作日内处理；涉及安全、风控、支付拒付、上游服务商、隐私核验、监管要求、司法程序或复杂争议的事项可能需要更长时间。

对于严重违法违规、儿童安全、恶意软件、攻击、欺诈、盗刷、洗钱、套现、支付风险、制裁或出口管制风险、监管风险或上游强制要求，GGUUAI 可先行采取必要限制措施，再进行后续审核。
