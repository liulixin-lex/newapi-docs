# 计费规则

采用 1:1 汇率制，即 1 ¥ = 1 $ 额度。

总折扣 = 汇率折扣 * 分组倍率

## 汇率折扣

1 美元额度只需充值 1 元人民币。

## 分组倍率

根据不同分组设置不同倍率档位：

倍率 = 1 按官方价格进行消耗。

倍率 < 1 享折扣：例如 0.5 倍 = 5 折，即消耗同等数量 token，所需费用为原价费用的 0.5 倍。

倍率 > 1 需加价：1.5 倍 = 额外支付 50% 费用，即消耗同等数量 token，所需费用为基础费用的 1.5 倍。

## 海外模型

::: tip 计费口径
基准价格通过 1:1 汇率，换算为人民币，在人民币基准价格基础上再 × 分组倍率。
:::

例如：

| GPT-5.5 | 输入 | 输出 | 缓存读取 |
| --- | ---: | ---: | ---: |
| OpenAI官方 | 5$/M | 30$/M | 0.5$/M |
| 本站基准价格 | 5¥/M | 30¥/M | 0.5¥/M |
| 本站0.5倍率 | 2.5¥/M | 15¥/M | 0.25¥/M |
| 本站1.5倍率 | 7.5¥/M | 45¥/M | 0.75¥/M |

## 国产模型

::: tip 计费口径
基准价格 = 官方价格，在官方价格基础上再 × 分组倍率。
:::

例如：

| deepseek-v4-pro | 输入 | 输出 | 缓存读取 |
| --- | ---: | ---: | ---: |
| deepseek-v4-pro官方 | 3¥/M | 6¥/M | 0.025¥/M |
| 本站基准价格 | 3¥/M | 6¥/M | 0.025¥/M |
| 本站0.7倍率 | 2.1¥/M | 4.2¥/M | 0.0175¥/M |
