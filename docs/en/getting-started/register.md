# Create an Account

Sign-up page: {{SIGN_UP_URL}}

![Sign-up page showing email registration plus GitHub and LinuxDO options](/images/tutorial/account-register-01-sign-up-options.webp)

## Open the Sign-up Page

1. Open the sign-up page and register with an allowed email address.
2. If you have a GitHub account, you can also click `Continue with GitHub` at the bottom to start registration.
3. If you have a LinuxDO account, you can also click `Continue with LinuxDO` at the bottom to start registration.

## Method 1 (Recommended): Register with GitHub

1. Click `Continue with GitHub`.
2. Select the GitHub account you want to bind in the pop-up window and complete authorization.
3. After authorization succeeds, the system automatically creates the account and signs you in.

GitHub registration does not require an extra password. For future sign-ins, continue using the same GitHub account.

## Method 2: Register with LinuxDO

1. Click `Continue with LinuxDO`.
2. Select the LinuxDO account you want to bind in the pop-up window and complete authorization.
3. After authorization succeeds, the system automatically creates the account and signs you in.

## Method 3: Register with Email

1. Choose a custom username and password.
2. Enter your email address and receive the verification code.
3. Submit the form as prompted to complete registration.
