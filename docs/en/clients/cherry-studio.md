# Other Third-party Client Setup

::: tip Suitable Scenario
If you only want to use a chat client, you can start directly with Cherry Studio setup.
:::

## Cherry Studio

Cherry Studio is a visual AI client. It is suitable for beginners who do not want to write commands and only need everyday chat features.

## Download Link

[https://www.cherry-ai.com/download](https://www.cherry-ai.com/download)

## How to Use

1. Open Cherry Studio, click the settings button in the upper-right corner, and enter the `Model Services` page.

![Model Services page in Cherry Studio settings](/images/tutorial/cherry-studio-01-model-services.webp)

2. At the bottom of the model service list, click `Add` to create a new provider, then fill it in by referring to the screenshot.

![Adding a new provider at the bottom of the Cherry Studio Model Services list](/images/tutorial/cherry-studio-02-add-provider.webp)

3. Open the GGUUAI provider you just created and fill in the API configuration.

- API key: enter your GGUU API key.
- API address: `https://api.gguuai.com`

![GGUUAI provider API key and API address settings in Cherry Studio](/images/tutorial/cherry-studio-03-provider-settings.webp)

4. Click `Get model list`, then choose the model you want to use in the model window, such as `gpt-5.5`.

![Fetching the model list in Cherry Studio and selecting the gpt-5.5 model](/images/tutorial/cherry-studio-04-fetch-models.webp)

5. Click the `Test` button in the upper-right corner. When it shows `Connection successful`, you can use it.

![Successful connection test in Cherry Studio](/images/tutorial/cherry-studio-05-connection-test.webp)

6. Return to `Home`, choose a model in the upper-left corner, and start chatting.

![Cherry Studio home page with a model selected for chat](/images/tutorial/cherry-studio-06-select-chat-model.webp)
